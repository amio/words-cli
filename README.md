# words-cli [![npm-version][npm-badge]][npm-link]

Find words with wilcard.

## Install

```shell
npm install -g words-cli
```

## Usage

```
  USAGE

    ${bold('words')} [wildcard]

  EXAMPLE

    ${bold('words "*am"')}    Find words ending with 'am'
    ${bold('words "??am"')}   Find words ending with 'am' of length 4
    ${bold('words "*??am"')}  Find words ending with 'am' of length 4+
```
